# Welcome to the Vs. 480 Mod Repository

This is different from [Vs480.Engine](https://github.com/480ScratchBlox/Vs480.Engine) because that hosts the engine while this hosts the mod itself.

# Installing

To install Vs. 480, clone this repo into your compiled [Vs. 480 Engine](https://github.com/480ScratchBlox/Vs480.Engine). If you can't compile, you can use the latest version of [Psych Engine](https://github.com/ShadowMario/FNF-PsychEngine/releases)

Basically, [follow this guide](https://youtu.be/f0u1F5YWQLg?si=QSrCqHYKVovy6Tc2) and you are good to go!
